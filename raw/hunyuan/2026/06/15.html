计费与结算方式

混元生文结算顺序为：赠送的免费资源包 > 付费资源包 > 后付费。
按用量（token 数）计费，包括直接调用 API 和在控制台 API Explorer 使用混元生文，调用失败不计费。
默认情况下，免费资源包和付费资源包耗尽或到期后，不会自动转入后付费。如需使用后付费模式结算，请前往腾讯混元大模型 控制台 > 设置 > 后付费设置 开通后付费。

费用计算示例

后付费

用户当月首次使用，累计调用混元生文接口 hunyuan-TurboS 系列模型共200万 tokens，则在使用达到100万 tokens 时消耗完免费资源包，剩余100万 tokens 需通过后付费进行结算，区分输入和输出。假设用户使用了80万 tokens 输入，20万 tokens 输出，所需支付的费用计算如下：
800000 tokens / 1000000 tokens × 0.8元/每百万 tokens + 200000 tokens / 1000000 tokens × 2元/每百万 tokens = 1.04（元）

预付费

当用户购买预付费点数资源包，假设用户使用 hunyuan-TurboS 系列模型的100万 tokens 输入+10万 tokens 输出并且还使用了 hunyuan-vision 的100万 tokens 输入+10万 tokens 输出，消耗点数计算如下：
(1000000 tokens / 1000 tokens x 0.08点/千 tokens + 100000 tokens / 1000 tokens x 0.2点/千 tokens) + (1000000 tokens / 1000 tokens x 1.8点/千 tokens + 100000 tokens / 1000 tokens x 1.8点/千 tokens) = 2080（点）

并发增购

用户当月有大流量，需要10并发，同时已经消耗完免费资源包，均使用后付费进行结算。当月累积调用混元生文接口 hunyuan-TurboS 系列模型，输入2000万 tokens ，输出6000万 tokens ，所需支付的费用计算如下：
预付费：提前增购并发 （ 10 - 5 ）并发× 800元/并发 = 4000（元） ，注：混元生文 hunyuan 接口默认限制为5个并发
后付费：消耗 token 费用 2000万 tokens / 1000000tokens × 0.8元/每百万 tokens + 6000万 tokens / 1000000 token × 2元/每百万 tokens = 136（元）
当月合计消费 4000 + 136 = 4136（元）
