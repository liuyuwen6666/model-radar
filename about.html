关于 AI 模型价格雷达

追踪每日主流大模型 API 价格、上下文与价格趋势，助开发者更低成本地打造 AI 应用。

我们的愿景

在 AI 时代，大模型的 API 价格、计费模式和上下文窗口大小每天都在发生翻天覆地的变化。各厂商为了市场竞争不断调整售价，并新增了类似上下文缓存（Session Cache）等高级计费策略。这使得开发者和企业在做架构与成本选型时面临巨大的信息壁垒。

AI 模型价格雷达 (ModelRadar) 旨在打破这个壁垒。我们致力于打造一个高可用、高度结构化、机器可读且每日实时更新的价格雷达，帮助全球开发者同屏比较模型调用成本，透明展示各大模型厂商的历史调价细节，帮助企业做出更有前瞻性的 AI 技术选型决策。

数据是如何获取的？

  • 动态抓取：平台自主开发了针对各大主流云厂商及模型提供商（如 OpenAI, Anthropic, Google, DeepSeek, 字节跳动, 腾讯, 阿里巴巴, 月之暗面等）定价页面的动态解析脚本。
  • 每日自动同步：每日通过自动化流程定期抓取官网最新的价格表，经过结构化映射与汇率换算后，自动生成全网统一的 models.json 与 changelog.json 数据。
  • 防挂兜底机制：为了应对云厂商页面结构突变引起的解析挂死，爬虫内置了极其稳健的最新实时 Fallback 蓝图数据集，确保在极端解析失败的情况下，依然能保障数据服务的连续性。
